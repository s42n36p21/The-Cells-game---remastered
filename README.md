# The-Cells-game---remastered

## Управление 
- WASD/URDL - ходьба
- Колесико мыши - zoom
- ЛКМ - взаимодействие с миром
- K/L - фуллскринн
- B - сменить фон
- V - сменить тип датчика
- С - on/off анимацию цепной реакции
- X - on/off звук
- E - включить режим редактирования
  - 1 - Создать клетку 
  - 2 - Удалить клетку 
  - 3 - Связать клетку 
  - 4 - Отвязать клетку 
  - 5 - Гибкая связка 
  - 6 - Отчистить связи 
  - 7 - Авто-связка 
  - 8 - Входящии связи 
  - 9 - Выходящие связи 
  - 0 - Тип клетки
- Z - сдаться во время игры
- Q - Новые игроки
- P - сохранить карту
- F - Free cam
- Y - Принудительный запуск симуляции

## Виды клеток
1. Базовая (Классическая) Клетка
2. Закрытая клетка
3. Клетка пустоты
